SuppressWarnings Overrides the method in the base class to suppress warnings - it can be set using the system property NExcel.nowarnings. This method was originally present in the WorkbookSettings bean, but has been moved to the logger class. This means it is now present when the JVM is initialized, and subsequent to change it on a Workbook by Workbook basis will prove fruitless
