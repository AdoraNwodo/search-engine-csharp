Contact Us

If you find a bug, please report it. Your help is welcome and will give a better library to everyone.

To report bugs use the Bug Tracker in SourceForge's NExcel site. Please don't send mails.

In order to ensure a correct and fast solution, write in Bug Tracker a detailed description of the problem, and attach a succint test case which can easily reproduce the problem. A simple test project in VS.NET or SharpDevelop is accepted.

Thanks for your help.
